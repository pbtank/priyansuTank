Abstract

The goal of his study project is to understand how actual flow of data happens to track the realtime position of a satellite in orbit. The data is fetched from official agencies like SpaceTrack in the form of Two Line Elements and then is propagated in time using standard SGP4 model and various orbitals elements are calculated for the satellite. The data is then stored and can be visualized using a deployment on github pages.
